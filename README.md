## 📝 Sobre o Projeto Original

O **AFIIP-SINCRONIZADOR** foi concebido originalmente como o motor de retaguarda (backend/automação) para dar suporte técnico à estrutura da **Adiel Fontes Investigação e Inteligência Privada**. 

A ferramenta tinha como objetivo automatizar processos de sincronização e o tráfego estruturado de dados/logs entre ambientes, servindo como um estudo prático de manipulação de arquivos e automação utilizando o ecossistema .NET.

---

## 🛠️ Tecnologias Utilizadas na Época

- **Linguagem:** C# (.NET Framework)
- **IDE de Desenvolvimento:** Visual Studio (Solução unificada `.sln`)
- **Foco de Estudo:** Lógica de sincronização, concorrência e manipulação de diretórios.
